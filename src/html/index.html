I'm Stephen Fan, a web designer and front end developer in US. My expertise lies in designing and creating websites to improve user experiences.
Design
Minimalism
Going minimal is the ultimate sophistication in websites. I am devoted to convey most information by using least contents for the users. My design embraces simplicity on every level.
Responsive
My work consists of a mix of flexible grids and layouts, images and an intelligent use of CSS media queries. As the user switches from their laptop to other devices, the website should automatically switch to accommodate for resolution, image size and scripting abilities.
Aesthetic
Designing a beautiful web page can be tricky, but I'm here to help. I focus on high-quality images, modern typography and contrasting colors to create aesthetic user interfaces.
Animation
Animation is complex. I'm constantly looking at ways to optimise user experiences with different animations on text and images. I don't provide best ways to use animations, but only beautiful ways.
